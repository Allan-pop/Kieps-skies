Introduction to Modern CSS Layouts

This tutorial demonstrates three powerful CSS layout techniques:

  • Flexbox: One-dimensional layout method for arranging items in rows or columns
  • CSS Grid: Two-dimensional layout system for more complex designs
  • Media Queries: Technique to create responsive designs that adapt to different screen sizes

Resize your browser window to see how the layouts adapt to different screen sizes!

CSS Grid Layout

CSS Grid is designed for two-dimensional layouts - rows and columns together.

Basic Grid Layout

A simple 3x3 grid:

1
2
3
4
5
6
7
8
9
/* Basic 3x3 grid layout */
.basic-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 rows of 100px each */
  gap: 10px; /* Space between grid items */
}

Grid Template Areas

Using named grid areas to create a layout:

Header
Sidebar
Main Content
Footer
/* Layout using named grid areas */
.grid-areas {
  display: grid;
  grid-template-columns: 1fr 3fr; /* Sidebar takes 1/4, main takes 3/4 */
  grid-template-rows: auto 1fr auto; /* Header/footer auto-sized, main takes remaining space */
  grid-template-areas: 
    "header header"  /* Header spans both columns */
    "sidebar main"   /* Sidebar and main content side by side */
    "footer footer"; /* Footer spans both columns */
  gap: 10px;
  height: 400px;
}
/* Each item is assigned to its area */
.grid-header { grid-area: header; }
.grid-sidebar { grid-area: sidebar; }
.grid-main { grid-area: main; }
.grid-footer { grid-area: footer; }

Responsive Design with Media Queries

Media queries allow you to apply different styles based on device characteristics like screen width.

Responsive Layout Example

This layout changes based on screen size. Resize your browser to see the changes:

1
2
3
4
/* Mobile first approach - starts as a column */
.responsive-container {
  display: flex;
  flex-direction: column;
  gap: 10px;
}

/* Tablet: 768px and up - becomes 2 columns */
@media (min-width: 768px) {
  .responsive-container {
    flex-direction: row;
    flex-wrap: wrap;
  }
  .responsive-item {
    flex: 0 0 calc(50% - 5px); /* Two items per row */
  }
}

/* Desktop: 1024px and up - becomes 4 columns */
@media (min-width: 1024px) {
  .responsive-item {
    flex: 0 0 calc(25% - 7.5px); /* Four items per row */
  }
}

Responsive Grid Example

@media (min-width: 1024px) {

  }
}

Common Media Query Breakpoints

Standard breakpoints for different devices:

/* Small devices (phones, 576px and up) */
@media (min-width: 576px) { ... }

/* Medium devices (tablets, 768px and up) */
@media (min-width: 768px) { ... }

/* Large devices (desktops, 992px and up) */
@media (min-width: 992px) { ... }

/* Extra large devices (large desktops, 1200px and up) */
@media (min-width: 1200px) { ... }

/* For print */
@media print { ... }

/* For dark mode */
@media (prefers-color-scheme: dark) { ... }

/* For reduced motion preferences */
@media (prefers-reduced-motion: reduce) { ... }

Combining Flexbox, Grid, and Media Queries

A real-world example combining all three techniques:

Complete Page Layout

A responsive page layout that uses both flexbox and grid:

Header
Navigation
Card 1
Card 2
Card 3
Card 4
Sidebar
Footer
/* Mobile layout: stacked vertically */
.page-layout {
  display: grid;
  grid-template-areas:
    "header"
    "nav"
    "main"
    "sidebar"
    "footer";
  gap: 10px;
}

/* Cards use flexbox for layout */
.page-main {
  display: flex;
  flex-direction: column;
  gap: 10px;
}

/* Tablet layout: sidebar next to main */
@media (min-width: 768px) {
  .page-layout {
    grid-template-columns: 3fr 1fr;
    grid-template-areas:
      "header header"
      "nav nav"
      "main sidebar"
      "footer footer";
  }
  
  .page-main {
    flex-direction: row;
    flex-wrap: wrap;
  }
  
  .card {
    flex: 0 0 calc(50% - 5px); /* Two cards per row */
  }
}

/* Desktop layout: nav, main, sidebar in a row */
@media (min-width: 1024px) {
  .page-layout {
    grid-template-columns: 1fr 4fr 1fr;
    grid-template-areas:
      "header header header"
      "nav main sidebar"
      "footer footer footer";
  }
}
